

  响应状态字典,  规范:功能_代码
  
  

//  登陆响应码
      LOGIN_001:登陆成功
	  LOGIN_002:用户名或者密码不正确
	  LOGIN_003:用户已被所动
	  LOGIN_004:验证码错误

// 用户模块
	  USER_001:用户注册成功
	  USER_002:用户修改成功
	  USER_003:用户删除成功
	  USER_004:用户注册失败:用户名已经存在
	  USER_005:用户修改失败:用户名已经存在
	  USER_006:用户删除失败:该用户不存在
	  USER_007:用户注册失败:两次输入的密码不一致
	
// 通用
      UNKNOW_9999: 未知错误
